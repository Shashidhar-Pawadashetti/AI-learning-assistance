SLIDE TITLE: Technology Stack

• Frontend Development
  - React.js (v19): Component-based UI library
  - Vite: Fast build tool and development server
  - Tailwind CSS: Utility-first styling framework
  - Firebase SDK: Authentication and client-side services

• Backend Development
  - Node.js: Runtime environment
  - Express.js: Web framework for API handling
  - Mongoose: ODM for MongoDB interactions
  - JWT (JSON Web Tokens): Secure authentication

• Database & Storage
  - MongoDB Atlas: Cloud-based NoSQL database

• AI & Machine Learning Services
  - Hugging Face API: Large Language Models (LLMs) for quiz generation
  - Tesseract.js: OCR for extracting text from images
  - PDF-Parse / Mammoth: text extraction from documents

1. SOFTWARE REQUIREMENTS
------------------------
To develop and run the "AI Learning Assistant", the following software environment is required:

| Component | Requirement | Recommended / Version |
|-----------|-------------|-----------------------|
| Operating System | Windows, macOS, or Linux | Windows 10/11, macOS Ventura+ |
| Runtime Environment | Node.js | v16.0.0 or higher |
| Package Manager | npm or yarn | npm v8+ |
| Database System | MongoDB | MongoDB Atlas (Cloud) |
| Code Editor | VS Code, Sublime, or WebStorm | VS Code (latest) |
| Web Browser | Chrome, Firefox, Edge, Safari | Chrome (latest) |
| API Testing | Postman or Insomnia | Postman |
| Version Control | Git | Git v2.3+ |

2. HARDWARE REQUIREMENTS
------------------------
The following hardware specifications are recommended for the development and deployment environment:

• Processor (CPU):
  - Minimum: Dual-core processor (Intel i3 or equivalent)
  - Recommended: Quad-core processor (Intel i5/Ryzen 5 or higher) for smoother AI processing and compilation.

• Memory (RAM):
  - Minimum: 4 GB
  - Recommended: 8 GB or higher (Node.js and local React servers can be memory intensive).

• Storage:
  - Minimum: 1 GB free disk space (for project files and dependencies).
  - Note: Additional space required for database if running MongoDB locally.

• Network:
  - Stable broadband internet connection is mandatory for:
    - Fetching data from MongoDB Atlas.
    - Communicating with Hugging Face and Google Gemini AI APIs.
    - Firebase Authentication services.

3. TECHNOLOGY STACK SPECIFICATIONS
----------------------------------
A detailed breakdown of the technologies used in the project:

3.1 Frontend
- Library: React.js (v19)
- Routing: React Router DOM
- Styling: Tailwind CSS
- Build Tool: Vite
- PDF Rendering: PDF.js

3.2 Backend
- Server: Node.js with Express.js
- Authentication: JSON Web Tokens (JWT) & Firebase Admin SDK
- File Handling: Multer (uploads), PDF-Parse, Mammoth (.docx)
- Security: Helmet, Cors, Express-Rate-Limit
- Email Services: Nodemailer (with SendGrid or Gmail)

3.3 Artificial Intelligence Integration
- Text Generation: Hugging Face Inference API (Meta-Llama, etc.)
- Visual Recognition: Tesseract.js (Optical Character Recognition)
- Alternative AI: Google Generative AI (Gemini)
